# Flood Segmentation UNet++ Checkpoint

This repository documents the Kaggle checkpoint file used for the AISEHack 2026 flood segmentation workflow.

## Dataset

- Dataset: Flood Segmentation UNet++ Model (AISEHack 2026)
- Kaggle URL: https://www.kaggle.com/models/hemantpande05/flood-segmentation-unet-model-aisehack-2026
- Author: Hemant Pande05
- Primary checkpoint file: s3_effb4_final.pth (84.53 MB)
- Task: 3-class semantic segmentation (Flood / Water / No-Flood)
- Model family: UNet++ with EfficientNet-B4 encoder
- License: CC0 1.0 (Public Domain)
